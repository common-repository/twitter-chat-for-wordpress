Beautiful Client Experience

Itweetlive free live chat widget connects your visitors to twitter in seconds.Whenever your visitors type in the widget area, it's immediately convertrd into a tweet sent to your twitter account!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `tchat.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In your WordPress: Admin Panel - Settings - Twitter Options: enter your Twitter user name
4. Now login to http://itweetlive.com.  When you login, your twitter icon will appear on your Chat Widget.
